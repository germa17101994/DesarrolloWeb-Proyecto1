El objetivo de este proyecto era un construir un Api construida con .NET Core que posteriormente iba a ser consumida por Angular para generar la vista. Se trabajo en un modelo MVC para implementarlo, el servidor de aplicaciones estaba alojado en un servidor de Azure y como plus del proyecto era implementar integración continua por medio de Azure DevOps tener un integracion continua.

Este proyecto tenia como objetivo, sustituir la forma de generar cheques ya que la empresa lo hacia manualmente. La idea era manejar un software que tuviera un interfaz para manejar todos lo relacionado con aprobar cheques. El software quue se utilizo para la interfaz fue por medio de web, la parte del backend se manejo con php y la conexion de datos (BD) se uso Oracle 11g. El fuerte de este proyecto era crear una base de datos solida para el resguardo de la informacion, en ellos se manejaron diferentes procedimientos como el uso de Constraints, Index, Triggers, Functions y Store Procedure.

Se requeria desarrollar un sistema electoral basado en la necesidad del problema que surgio en el 2018 con los datos que no coincidian, entonces se deberia de implementar la seguridad de la informacion para que fuera transparente. El proyecto se requeria trabajarlo en modelo MVC, por lo cual se trabajo con Java, Spring y Hibernate. Se desarrollo documentacion necesaria para modelar el proyecto, en pocas palabras se hizo un analisis y diseño del software para posteriormente desarrollarlo.
